Installer PicModifier
======================
Aucune installation n’est nécessaire. Il suffit d’aller dans le dossier Executable pour trouver une version fonctionnel du logiciel.

Installation pour le dévellopement de PicModifier
==================================================
 1- Installer Visual Studio 2010
 2- Télécharger Qt 4.8.4 pour Visual studio 2010 au lien suivant:  http://download.qt-project.org/official_releases/qt/4.8/4.8.4/qt-win-opensource-4.8.4-vs2010.exe
 3- Télécharger le Add-in de Qt4 pour Visual Studio au lien suivant : http://download.qt-project.org/official_releases/vsaddin/qt-vs-addin-1.1.11-opensource.exe
 4- Installer Qt 4.8.4 précédemment téléchargé
	4.1- Accepter les conditions et les paramétrages par défaut
 5- Installer le Add-in pour Visual studio précédemment téléchargé
 6- Ouvrir la solution du projet et compiler
